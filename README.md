# **Machine Learning Regression Projects**

This repository is a collection of applied Machine Learning projects focused on regression models, with an emphasis on **understanding, implementation, and experimentation**.

The goal is not only to use models, but to deeply understand their behavior, assumptions, and limitations through hands-on practice.

---

## Objectives

* Build strong intuition behind regression algorithms
* Apply models to real-world datasets from Kaggle
* Perform proper data preprocessing and feature engineering
* Analyze model performance
* Compare models and understand when to use each one

---

## Repository Structure

Each project is implemented as a detailed Jupyter Notebook that includes:

* Problem definition
* Data exploration (EDA)
* Data preprocessing
* Model implementation
* Evaluation and analysis
* Key insights and conclusions

---

## Models Covered

* Multiple Linear Regression
* Polynomial Regression
* Support Vector Regression (SVR)
* Decision Tree Regression
* Random Forest Regression
